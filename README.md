# ThadMob Support

ThadMob is a youth soccer team management app for iOS.

## Contact

For support, bug reports, or feature requests, please [open an issue](https://github.com/JonnyChappp/thadmob-support/issues) on this repository.

## Features

- Squad management
- Match logging & player ratings
- Player stats tracking
- League integration (PlayMetrics)
- GPS heatmaps (Apple Watch & Garmin)
- Manager & player modes

## Privacy

ThadMob uses Firebase Authentication for sign-in and Firestore for data storage. GPS data is imported only when you choose to do so from Apple Watch workouts or Garmin GPX files. No data is shared with third parties.
